Dataflow Visualization Builder

Introduction

Dataflow Visualization Builder is a tool that enables creation of visualization systems for multi-dimensional table data.

Visualization system is generated by dataflow diagram specification, which defines the query and filter logic inside the system.

Dataflow Diagram

Dataflow diagram consists of nodes that process/filter the data and connections that link the nodes.

The diagram is a directed acyclic graph in a topological sense.

DataSet and ValueSet

There are two types of sets passed between dataflow nodes: DataSet and ValueSet.

DataSet is essentially a subset of rows of the loaded table data.

ValueSet is a list of constants for filtering purpose.

Node

A node is a stand-alone componenent for a specific task. For example, data source node loads the data and visualization node presents a plot of the data.

Port

Each node will have ports located on its sides. The input ports on the left of a node defines the input a node accpets, e.g. data, filter parameters. The output ports on the right of a node defines what a node produces for the downflow node.

A port may either be single or multiple. A single port can make at most one connection. A single port has a single dot under the port letter, while a multiple port has three.

A data port is indicated by letter D. A value port is indicated by letter V. A selection port is indicated by letter S (selected items in visualization).

Connection

A connection passes the dataset/valueset from its source node to its target node.

Diagram Design

Creation

To create a node, select Add in the menu and choose a node type. The node creation panel can be called using shortcuts (A or shift+A).

To create a connection, drag from an output port to an input port. Alternatively, you may drop a connection onto the center of the node to make a connection to the first connectable port.

Editing

Drag the canvas to pan.

You can drag a node to re-position it.

Hold Ctrl to drag a box and range select several nodes at a time. All selected nodes can be interacted with together.

Right-click on nodes or edges to view menu options.

Visualization nodes block interactions in its rendering area (svg). Hold Ctrl when dragging to force interacting with underlying node.

Shortcuts

System

M toggle menu panel

ctrl+S save dataflow

ctrl+L load dataflow

Diagram Editing

A call the add node compact panel

Shift+A call the add node full panel

(when add node panel is open) press the first letter of node/group names to filter

D toggle detailed display of node

T toggle display of node options (e.g. dimension selection)

Shift (hold) add node(s) to the current node selection

Delete or ctrl+X delete the selected node(s) / hovered edge

Right-click (on ports) delete all the connections from this port

Visualization

Ctrl+A select all items in the node

Ctrl+Shift+A de-select all items in the node

Shift (hold) add items to the current item selection
